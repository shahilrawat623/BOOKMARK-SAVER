# BOOKMARK-SAVER
📑 Bookmark Saver A simple, lightweight web application to save, manage, and access your favorite websites. Built with HTML, CSS, and JavaScript, it uses localStorage to persist bookmarks directly in your browser—no backend required.

🛠️ Tech Stack
- HTML5 → Structure

- CSS3 → Styling 

- JavaScript (ES6) → Logic & localStorage

  📂 Project Structure

  bookmark-saver/
│── index.html      # Main HTML file
│── style.css       # Styling
│── script.js       # Core logic
│── README.md       # Documentation
